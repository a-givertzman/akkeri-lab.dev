FRDM — Система мониторинга каната грузоподъемных механизмов

Фиксирует наработку каната по всей его длине с учетом текущей нагрузки. Помогает оператору оценить степень износа отдельных участков каната. Фиксирует повреждения каната по видео с нескольких кмер, установленных по периметру каната. Области применения:
  • Промышленные грузоподъемные механизмы;
  • Грузоподъемные механизмы грузовых судов.

    • Для получения информации о способах приобретения права использования наших продуктов и стоимости их использования свяжитесь с нами по почте: info@sa-lab.dev

    14.01.2025 | Начало разработки

      Функциональные возможности
    • Для стального каната
    • Для оптоволоконного каната (планируется поддержка)
    • Способы диагностики
      • По количеству проходов через блок c учетом нагрузки
      • По изображению с видеокамер
        • Обработка изображений с нескольких камер одновременно
        • Двухступенчатый алгоритм детектирования повреждений
        • Первая ступень: Быстрый предварительный анализ по контуру каната
        • Вторая ступень: Улучшенный анализ по цветной гистограмме (Метод Отсу)

    15.01.2025 | Главный экран

    • Отображение состояния каната на столбчатой диаграме
    • Ось X - вся длина каната
    • Ось Y - степень наработки участка каната с учетом нагрузки
    • Индикаторы срабатывания видео-детектора под осью X
